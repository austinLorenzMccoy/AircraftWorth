## 🎯 Judge Questions & Answers

### **"Is this running on live data?"**
**Answer:** "Yes - HCS topic 0.0.7968510 is live on Hedera testnet. Every MLAT position is logged there. Click any aircraft and hit HashScan to verify. If we're in replay mode due to testnet connectivity, live is one env var away."

### **"How is this different from FlightAware?"**
**Answer:** "FlightAware requires ADS-B transponders. We track aircraft with no transponder at all, using time-difference-of-arrival across 12 distributed sensors. Ghost flights are no longer invisible."

### **"What does the AI actually do?"**
**Answer:** "Groq's llama-3.3-70b runs at 200ms - fast enough to annotate every MLAT fix as it arrives. It classifies track patterns: evasive maneuvers, altitude deviations, holding patterns. Then judges can query in plain English."

### **"Where are the smart contracts?"**
**Answer:** "AircraftWorthMarketplace, EscrowService, and ReputationSystem - all deployed to Hedera testnet. The marketplace UI uses them for trustless escrow. The escrow literally cannot release without an HCS sequence number as delivery proof."

### **"How does Hedera help vs just Ethereum?"**
**Answer:** "HCS is 50ns-precision timestamping at $0.0001 per message. Our MLAT timestamps need nanosecond accuracy. No other chain gives us that. HTS gives us NFTs for position proofs. The smart contracts run on Hedera EVM."
